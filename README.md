# 📅 Interactive Calendar Component

A modern, responsive wall calendar built using Next.js and Tailwind CSS.  
This project is inspired by a physical wall calendar design and enhanced with interactive features.

---

## 🚀 Features

### 📌 Core Features
- 🖼️ Wall calendar UI with hero image
- 📅 Dynamic monthly calendar grid
- 🔁 Month navigation (previous / next)
- 🟦 Day range selection (start → end)
- 🎯 Highlight for today’s date

### 📝 Notes Section
- ✍️ Editable notes panel
- 💾 Auto-save using localStorage
- 📓 Notebook-style UI

### 📱 Responsive Design
- 💻 Desktop: Side-by-side layout (Notes + Calendar)
- 📱 Mobile: Stacked layout for touch usability

---

## 🛠️ Tech Stack

- Next.js
- React (Hooks)
- Tailwind CSS
